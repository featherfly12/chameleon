# chameleon lamp
making chameleon lamp so that it senses the color and turns into that color!\
you can find the code, cad, and hand drawn/tinkercadded wire diagrams here!
made it for my little sister who is scared of the dark and has a obsession with lamps\
using a FIRST Robotics REV color sensor :skull: to sense as a challenge\
:)


how to assemble \ 
1. Put the electronics together based on the schematic (not a pcb)\
2. put the electronics into the box\
3. glue/attatch the branch of the lamp to the box\
4. if you didn't alr, wrap the lamp in lamp fabric/fabric\
6. wire the led up into the branches of the lamp\
7. attatch to power source and upload code through arduino\
8. to upload coade, just down load the adruino file, connect to your uno, and compile and run!\
9. test out the sensor!!! watch the lights change color!
